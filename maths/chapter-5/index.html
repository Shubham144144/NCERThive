About Chapter 5 – Arithmetic Progressions

Key Formulas – Chapter 5 Arithmetic Progressions

  • General form of AP: a, a+d, a+2d, a+3d, ...
  • nth term: aₙ = a + (n–1)d
  • Sum of n terms: Sₙ = n/2 [2a + (n–1)d]
  • Sum using last term: Sₙ = n/2 (a + l), where l = last term
  • Common difference: d = a₂ – a₁
  • Number of terms: n = (l – a)/d + 1

Important Points to Remember

  • In an AP, the difference between any two consecutive terms is always the same.
  • Common difference d can be positive, negative, or zero.
  • If d = 0 → all terms are equal (constant AP).
  • If d > 0 → AP is increasing.
  • If d < 0 → AP is decreasing.
  • Three numbers a, b, c are in AP if b – a = c – b → 2b = a + c.
